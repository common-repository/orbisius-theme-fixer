This plugin allows you to recover your site if a theme crashed it.
When the plugin is active it will reset the theme and you'll.

== Installation == 
You will have to upload this plugin into wp-content/mu-plugins folder.
If it doesn't exist create it.
Then the plugin will take over and will use one of the default WordPress themes.
